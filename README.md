## Практическая работа 1 — CSS-препроцессоры

**Задание:**  
Разработать карточку товара с использованием SASS (переменные, миксины, вложенность).

**Результат:**  
Готовая страница с тремя карточками товаров для магазина игрушек.  
Файл: `Practice1/index.html`


## Практическая работа 2 — Express.js API

**Задание:**  
Реализовать REST API для управления товарами.

**Результат:**  
Сервер на Express с CRUD-операциями. Доступные маршруты:
- `GET /products` — список товаров
- `GET /products/:id` — товар по ID
- `POST /products` — создание товара
- `PATCH /products/:id` — обновление товара
- `DELETE /products/:id` — удаление товара


## Практическая работа 3 — JSON и внешние API

Протестировать API с помощью Postman, изучить структуру JSON.

**Результат:**

- 5 запросов к внешнему API (JSONPlaceholder)

## Практическая работа 4 — API + React

**Задание:**  
Разработать интернет-магазин на React, подключить его к Express API.

**Результат:**  
- Бэкенд: Express, 9 товаров, CORS
- Фронтенд: React, компоненты, состояние
- CRUD: просмотр, добавление, редактирование, удаление товаров
- Разделение по категориям (Для собак / Для кошек)


## Практическая работа 5 — Swagger документация

**Результат:**  
- Сервер с пользователями (users) и CRUD-операциями
- Полная OpenAPI-спецификация
- Интерактивный интерфейс Swagger UI по адресу `/api-docs`
