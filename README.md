## Практическая работа 2 — Express.js API

**Задание:**  
Реализовать REST API для управления товарами.

**Результат:**  
Сервер на Express с CRUD-операциями. Доступные маршруты:
- `GET /products` — список товаров
- `GET /products/:id` — товар по ID
- `POST /products` — создание товара
- `PATCH /products/:id` — обновление товара
